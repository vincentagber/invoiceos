# 🖥️ invoiceOS Frontend

The frontend of invoiceOS is a modern, high-performance web application built with Next.js and the **Sovereign Clean** design system.

## 🎨 Design System
- **Aesthetic**: Premium Dark Mode, Glassmorphism, Fluid Transitions.
- **Components**: Custom UI library built on top of Tailwind CSS and Framer Motion.
- **Responsiveness**: Fully optimized for Desktop, Tablet, and Mobile.

## 🛠 Features
- **App Router Architecture**: Leverages Next.js 16 layouts and server components.
- **Real-time Analytics**: Interactive charts using Recharts.
- **PDF Generation**: In-browser invoice generation via jsPDF.
- **State Management**: Lightweight and fast global state with Zustand.

## 🚀 Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
